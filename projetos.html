Testes automatizados com Cypress

Condução de testes abrangentes, incluindo testes funcionais, de integração, de interface gráfica do usuário (GUI) e de API utilizando a linguagem JavaScript.

Os testes foram realizados na plataforma Central de Atendimento ao Cliente da escola Talking About Testing, denominada CAT TAT.
